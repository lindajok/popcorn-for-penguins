# Run Flask:
1. Activate environment: . demoenv/bin/activate
2. Install Flask if not there: pip3 install Flask
3. Setup: export FLASK_APP=search_engine.py, export FLASK_ENV=development, export FLASK_RUN_PORT=8000
4. Run: flask run
